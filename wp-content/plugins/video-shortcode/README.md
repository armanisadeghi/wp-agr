## Description ##

Add button on WordPress Editor that allows to add oEmbed videos supported by WordPress. Ability to add thumbnail preview for your video.

## Installation ##

### Manual Installation ###

1. Upload the entire `/video-shortcode` directory to the `/wp-content/plugins/` directory.
2. Activate Video Shortcode through the 'Plugins' menu in WordPress.
